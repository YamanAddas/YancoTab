# Changelog

## v2.1.0 — 2026-04-10

Initial release of YancoTab.

- 18 apps: Browser, Notes, Todo, Pomodoro, Calculator, Weather, Clock, Files, Settings, Solitaire, Spider Solitaire, Minesweeper, Mahjong, Snake, Memory, Tic-Tac-Toe, Tarneeb, Trix
- Chrome Extension (MV3) with new tab override
- Unified storage layer with cross-device sync via chrome.storage.sync
- Export/Import data with backup and restore
- Service worker for offline support (standalone mode)
- Cosmic glass theme with starfield backgrounds and 10 wallpapers
- Landing page
